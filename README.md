# שיפוצים לי

אתר תוכן לשיפוצים בעברית — Next.js (App Router) + TypeScript + Tailwind + MDX. סטטי לחלוטין (SSG), מותאם SEO ומנועי AI.

## הרצה מקומית

```bash
npm install
npm run dev
```

## פריסה (GitHub + Vercel)

1. `git init && git add -A && git commit -m "initial"` ודחפו לריפו GitHub
2. ב-Vercel: New Project → Import מהריפו. אין צורך בהגדרות מיוחדות
3. אחרי חיבור דומיין, עדכנו את `site.url` ב-`src/config/site.ts`
